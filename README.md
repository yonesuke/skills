# Skills Repository

Collection of agentic skills for autonomous coding tasks, following the [Skill Engineering Standard](AGENTS.md).

## Available Skills

| Skill | Description |
| :--- | :--- |
| [**JAX**](jax/SKILL.md) | Essential tools for using JAX in machine learning and mathematical analysis. |
| [**Flax Models**](developing-flax-models/SKILL.md) | Comprehensive guide for developing neural networks using Flax NNX. |
| [**Linear Solver**](linear_solver/SKILL.md) | Solves linear systems (Ax=b) using NumPy, JAX, PyTorch, and Lineax. |
| [**Option Pricing**](option_pricing/SKILL.md) | Option pricing tools including Black-Scholes and Greeks calculation. |
| [**Polars**](polars/SKILL.md) | High-performance DataFrame library usage. |
| [**UV**](uv/SKILL.md) | Fast Python package installer and project manager. |
| [**Interpolation**](interpolation/SKILL.md) | Techniques for 1D and 2D data interpolation (SciPy, JAX). |
| [**mHC**](mhc/SKILL.md) | Manifold-Constrained Hyper-Connections for deep transformers. |

## Usage

Each skill directory contains:
- `SKILL.md`: Entry point and API surface.
- `examples.md`: Copy-pasteable code patterns.
- `scripts/`: Executable ground-truth Python scripts.
- `reference.md`: Deep dives and theoretical background.

Refer to [AGENTS.md](AGENTS.md) for the complete engineering standard.
